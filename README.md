# 極致效能互動式個人網站 (Interactive Personal Website)

![Status](https://img.shields.io/badge/Status-In--Development-orange)
![Version](https://img.shields.io/badge/Version-0.1.0-blue)
![Stack](https://img.shields.io/badge/Stack-Vite%20%7C%20TS%20%7C%20Rust--Tools-green)

這是一個由 **AI 資深架構師與設計師** 打造的頂級個人網站專案。我們不僅追求視覺上的驚艷（WOW factor），更在工程底層導入了 2026 年最頂尖的開發工具鏈。

## ✨ 核心特色

- **極致效能**：基於 Vite 與 Rust 工具鏈（uv, ruff, biome），實現毫秒級的建置與載入。
- **頂級美學**：由 30 年經驗設計師指導，專注於微動畫、玻璃擬物化（Glassmorphism）與動態排版。
- **AI 驅動開發**：內建 `Agents.md` 規範，完美適配所有現代 AI 編碼助手。
- **全自動化工作流**：使用 Python (`uv`) 處理所有重複性勞動。

## 🚀 快速啟動

### 前置需求
- [uv](https://github.com/astral-sh/uv) (Python 管理)
- [Node.js](https://nodejs.org/) (LTS)
- [GitHub CLI](https://cli.github.com/)

### 安裝步奏
1. **初始化環境**
   ```bash
   uv sync
   npm install
   ```

2. **啟動開發伺服器**
   ```bash
   npm run dev
   ```
